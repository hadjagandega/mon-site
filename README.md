# Site Personnel

Un site web personnel moderne et responsive créé avec HTML5, CSS3 et JavaScript.

## 🚀 Fonctionnalités

- **Design moderne** avec Tailwind CSS
- **Responsive** sur tous les appareils
- **Navigation fluide** avec défilement doux
- **Formulaire de contact** interactif
- **Animations** et transitions élégantes
- **Menu mobile** adaptatif

## 📁 Structure du projet

```
site-personnel/
├── index.html          # Page principale du site
└── README.md          # Documentation du projet
```

## 🛠️ Technologies utilisées

- **HTML5** - Structure sémantique
- **Tailwind CSS** - Framework CSS moderne
- **JavaScript** - Interactivité et animations
- **Font Awesome** - Icônes professionnelles
- **Google Fonts** - Typographie moderne (Inter)

## 🎨 Sections du site

1. **Accueil** - Hero section avec présentation personnelle
2. **À propos** - Parcours et réalisations
3. **Compétences** - Technologies et domaines d'expertise
4. **Projets** - Portfolio avec exemples de travaux
5. **Contact** - Formulaire de contact et coordonnées

## 📱 Personnalisation

### Modifier les informations personnelles

1. Ouvrir `index.html`
2. Remplacer "Votre Nom" par votre véritable nom
3. Mettre à jour les informations de contact :
   - Email : `votre.email@example.com`
   - Téléphone : `+33 6 12 34 56 78`
   - Localisation : `Paris, France`

### Personnaliser les projets

Modifier la section "Mes projets" pour ajouter vos propres réalisations :
- Titre du projet
- Description
- Technologies utilisées
- Liens vers la démo et le code source

### Adapter les compétences

Mettre à jour la section "Mes compétences" avec vos technologies préférées et votre niveau d'expertise.

### Modifier les couleurs

Le thème utilise principalement la couleur violette (`purple-600`). Pour changer :
- Modifier les classes `purple-600` par la couleur de votre choix
- Adapter les dégradés CSS dans la section `<style>`

## 🌐 Déploiement

### Option 1 : GitHub Pages

1. Créer un repository GitHub
2. Uploader les fichiers du projet
3. Activer GitHub Pages dans les settings du repository
4. Le site sera disponible à `https://votre-username.github.io/nom-du-repo`

### Option 2 : Netlify

1. Créer un compte Netlify
2. Glisser-déposer le dossier du projet
3. Le site sera déployé instantanément avec une URL personnalisée

### Option 3 : Hébergement local

Ouvrir simplement le fichier `index.html` dans votre navigateur web.

## 🔧 Maintenance

- Le site est statique, aucune maintenance requise
- Pour ajouter de nouvelles fonctionnalités, modifier directement le HTML/CSS/JS
- Les dépendances sont chargées via CDN, pas d'installation nécessaire
